As principais tags

No HTML - Linguagem de Marcação de Hipertexto, temos diversas tags que agora vamos explorá-las!

  1. h1 até h6: são utilizadas para criar cabeçalhos ou títulos em uma página web, sendo de forma decrecente (h1 menor para h6 menor)
  2. P: a letra p é usada para definir um parágrafo.
  3. Mark: usado para fazer um: marca texto.
  4. Small: deixa o formato do: texto reduzido.
  5. U e I: respectivamente a função deles é sublinhado e itálico.
  6. Strong: texto fica em: negrito
  7. ol, ul e li: ol cria uma lista não ordenada sendo representado por um (•), diferente da ul sendo representada numericamente como esta lista.
    ambas precisam de um li para funcionar.
  8. blockquote: é utilizado para colocar citações em sua página
  9. Sup e sub: respectivamente acima da linha da base enquanto em baixo da linha da base
  10. hr: cria uma linha na página como abaixo:

Finalizando temos o < A >: entretanto ele tem uma direfenças dos demais, está tag necessita de um atributo (href""), sua função é direcionar para outros link/página.

tags extras

  1. Font, mudar a fonte do: texto (Arial) atributo face para funcionar
  2. Del, representar texto deletado ou removido de um documento. (somente um exemplo)
